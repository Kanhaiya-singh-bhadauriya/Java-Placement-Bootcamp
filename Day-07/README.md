# 📅 Day 07 - Arrays (Part 7)

## 📖 Overview

Day 07 focused on solving array problems using **nested loops**, **boolean flags**, and **algorithmic thinking** without using `HashMap`, `ArrayList`, or any Collection Framework.

The main objective was to understand how to identify previously processed elements, count frequencies, find unique values, and remove duplicate elements using only arrays.

---

# 📂 Programs Completed

## 1️⃣ Find Frequency of Each Element

### 🎯 Objective
Find the frequency of every distinct element in an array and print each element only once.

### 🧠 Algorithm
1. Read all elements into the array.
2. For every element, check whether it has already appeared before.
3. If already processed, skip it.
4. Otherwise, count its occurrences in the remaining array.
5. Print the element along with its frequency.

### 📊 Complexity
- **Time Complexity:** O(n²)
- **Space Complexity:** O(1)

---

## 2️⃣ Find Unique Elements

### 🎯 Objective
Print only those elements that appear exactly once in the array.

### 🧠 Algorithm
1. Read all array elements.
2. Skip elements already processed.
3. Count occurrences of the current element.
4. If its frequency is exactly one, print it.

### 📊 Complexity
- **Time Complexity:** O(n²)
- **Space Complexity:** O(1)

---

## 3️⃣ Remove Duplicate Elements

### 🎯 Objective
Remove duplicate elements and print only the distinct elements while preserving their original order.

### 🧠 Algorithm
1. Read all array elements.
2. Check whether the current element has already appeared.
3. If already processed, skip it.
4. Otherwise, print the element.

### 📊 Complexity
- **Time Complexity:** O(n²)
- **Space Complexity:** O(1)
